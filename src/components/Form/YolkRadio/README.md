## 单选框组件
### props
| 参数       | 说明         | 类型         | 默认值 |
| ------------ | -------------- | -------------- | ------ |
| outClassName | 最外层样式类名 | String         | -    |
| radioInfo    | 单选框的信息 | String / Array | null   |
| isChecked    | 当前是否勾选 | Boolean        | false  |
| disabled     | 是否禁用   | Boolean        | false  |
### Events
| 事件名称 | 说明       | 回调参数 |
| -------- | ------------ | -------- |
| click | 点击按钮事件 | 无      |
