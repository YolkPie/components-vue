## 底部按钮组件
### props
| 参数       | 说明                                | 类型  | 默认值 |
| ------------ | ------------------------------------- | ------- | ------ |
| bottomFit    | 底部是否自适应，针对iphoneX及以上机型 | Boolean | false  |
| outClassName | 最外层样式类名                 | String  | -     |
### Events
| 事件名称 | 说明       | 回调参数 |
| -------- | ------------ | -------- |
| clickBtn | 点击按钮事件 | 无      |
