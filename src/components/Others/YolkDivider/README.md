## Divider 分割线组件
### props
| 参数       | 说明         | 类型                  | 默认值 |
| ------------ | -------------- | ----------------------- | -------- |
| outClassName | 最外层样式类名 | String                  | -        |
| direction    | 单选框的信息 | String                  | vertical |
